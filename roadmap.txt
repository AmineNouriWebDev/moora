3. Design et Expérience Utilisateur (UI/UX)

•
Thème Principal : Thème clair (Light Mode) dominant, inspiré par des sites comme novago.com.tn et qjmotor.fr. Le design doit être épuré, professionnel, avec beaucoup d'espace blanc pour mettre en valeur les produits.

•
Fonctionnalité requise : Un bouton "Dark Mode" (Mode Sombre) accessible dans le header pour basculer l'interface.

•
Inspiration UI : Prendre exemple sur ki-motors.tn pour les options interactives fantastiques, notamment la possibilité de changer la couleur des motos directement sur la page (sélecteur de couleurs interactif).

•
Responsive Design : Le site doit être parfaitement adapté aux mobiles, tablettes et ordinateurs de bureau (Mobile-First approach).

3.1. Animations et Interactions (Inspirations)

Pour une expérience utilisateur dynamique et moderne, les animations suivantes sont requises :

•
Transitions de Page : Effets de fondu (fade-in) fluides lors du chargement des sections au défilement, inspirés par QJ Motor France.

•
Menu de Navigation :

•
Menu "sticky" (fixe) qui reste visible au défilement.

•
Soulignement animé des liens au survol (hover) avec une transition douce.

•
Menus déroulants (dropdowns) avec une légère transition de haut en bas.



•
Interactions Produits (survol) :

•
Légère élévation de la carte produit (box-shadow) et zoom discret sur l'image de la moto, similaire à QJ Motor France.

•
Effets de "glow" ou de changement de couleur sur les boutons "Coming Soon" pour les rendre interactifs.



•
Carrousels : Glissement fluide (swipe) entre les modèles phares dans le Hero Banner, avec des indicateurs de progression.

•
Animations au Défilement (Scroll Animations) : Apparition des éléments (icônes, texte, images) avec des effets de glissement (slide-up) ou de zoom-in progressifs, inspirés par Novago Tunisia. L'utilisation de bibliothèques comme AOS (Animate On Scroll) ou WOW.js est recommandée.

•
Boutons Flottants : Un bouton WhatsApp ou "Réserver un essai" fixe dans un coin de l'écran avec une petite animation de pulsation pour attirer l'attention (Novago Tunisia).

•
Sélecteur de Couleurs Dynamique : L'animation de changement d'image de la moto doit être instantanée mais fluide (fondu croisé) lors de la sélection d'une couleur.

•
Mode Sombre : Transition globale de la palette de couleurs (background et texte) avec une durée de 0.3s à 0.5s pour une expérience visuelle agréable.

•
Vidéo Background : Intégration de vidéos en arrière-plan (muted, autoplay, loop) dans le Hero Banner pour un aspect premium, comme observé sur QJ Motor France.

6. Instructions pour la Génération du Code

1.
HTML5 Sémantique : Utiliser des balises claires (<header>, <nav>, <section>, <article>, <footer>).

2.
CSS3 Moderne : Utiliser Flexbox et CSS Grid pour la mise en page. Implémenter des variables CSS (Custom Properties) pour gérer facilement le basculement entre le thème clair (par défaut) et le thème sombre.

3.
JavaScript (Vanilla) :

•
Créer la logique pour le bouton "Dark Mode".

•
Créer la logique pour le sélecteur de couleurs des motos (changer l'image affichée en fonction de la couleur sélectionnée par l'utilisateur).

•
Gérer le menu de navigation responsive (menu hamburger sur mobile).

•
Implémenter les animations au défilement (scroll animations) et les effets de survol (hover effects) décrits dans la section 3.1.



4.
Design System : Proposer une palette de couleurs élégante (dominante blanche/grise claire pour le fond, avec des accents de couleur pour les boutons et les éléments interactifs). Choisir des polices Google Fonts modernes (ex: Roboto, Montserrat ou Poppins).

Génère le code complet (HTML, CSS, JS) dans des blocs séparés, prêt à être intégré par un développeur. Assure-toi que le design soit à la hauteur des références fournies (Novago, QJ Motor, KIM Motors).
